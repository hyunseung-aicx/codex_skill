# Progress Directory

File-based state tracking for multi-session workflows.

## Structure
- `handoff-*.md` — Session handoff documents (auto-generated by `/handoff`)
- `feature-*.md` — Feature progress with DoD checklists
- `evals/` — Evaluation results and benchmarks

## Usage
- Run `/handoff` before ending a complex session
- New session: read latest `handoff-*.md` for context
- Feature tracking: create `feature-{name}.md` with machine-verifiable DoD
